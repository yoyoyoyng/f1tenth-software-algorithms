# Public Release Checklist

Before publishing this repository publicly, check the following items.

- [ ] 영상 또는 스크린샷에 개인정보, 계정명, 학번, 이메일 등이 노출되지 않는지 확인
- [ ] 실제 실행에 필요한 waypoint/map 파일을 공개해도 되는지 확인
- [ ] 코드 상단의 debug flag와 실행 경로가 공개용 설명과 일치하는지 확인
- [ ] ROS 2 package로 실행하려면 package structure 또는 launch file을 별도로 정리
